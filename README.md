# 四川麻将 · 血战到底

一个基于 React + TypeScript 的四川麻将游戏，支持血战到底玩法。

## 在线试玩

[https://mahjong.leoyuan.dev](https://mahjong.leoyuan.dev)

## 功能特性

### 游戏模式
- **普通模式**：血战到底，三家胡牌或流局结束，杠牌收分
- **手气模式**：胡牌即结束，纯娱乐不计分，带撒花动画

### 核心玩法
- 定缺：开局选择一门花色定缺
- 碰/杠：支持明杠、暗杠、补杠
- 胡牌：支持自摸、点炮
- 血战到底：一家胡牌后，其他家继续游戏
- 流局处理：查花猪、查大叫、退杠分

### 战绩系统
- 自动记录每局战绩
- 查看胡牌情况、番型、分数变化
- 支持清空战绩

### 其他功能
- 听牌提示
- 最后出牌高亮闪烁
- 响应式设计

## 技术栈

- React 19
- TypeScript
- Vite
- Zustand（状态管理）
- Vitest（测试）

## 游戏规则

### 定缺
- 开局时选择一门花色作为定缺
- 定缺花色的牌不能用于碰/杠（胡牌除外）
- 手牌中定缺花色必须全部打出

### 杠牌
- **明杠**：碰后加杠，或别人打出第四张，收 5 分
- **暗杠**：自己摸到四张，收 10 分
- **补杠**：碰后摸到第四张，收 5 分

### 胡牌
- **自摸**：每家付基础分 × 3
- **点炮**：点炮者付基础分

### 流局
- 牌墙摸完未结束则流局
- 查花猪：手牌还有定缺花色，赔给非花猪玩家
- 查大叫：未花猪且未听牌，赔给听牌玩家
- 退杠分：退还本局所有杠牌收分
